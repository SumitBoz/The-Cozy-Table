# The Cozy Table – Restaurant Management System
# Overview
The Cozy Table is a restaurant management application built using Angular, HTML, CSS, and TypeScript. It streamlines restaurant operations by providing features like user authentication, menu management, order processing, table reservations, and role-based access for admins and customers.

# Features
Authentication & Authorization – Secure login/signup for both admins and customers.
Menu Management – Add, update, and remove menu items dynamically.
Order Management – Place, track, and manage orders in real-time.
Table Reservation – Customers can book tables in advance.
Feedback System – Users can share reviews and feedback.
Role-Based Access – Different functionalities for admins and users to ensure security and efficiency.
